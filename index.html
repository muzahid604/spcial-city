how does javascript work?

জাভাস্ক্রিপ্ট হলো একটি জনপ্রিয় ক্লায়েন্ট সাইড স্ক্রিপ্ট ল্যাংগুয়েজ, এটা একটি interpreted language। এটা C এর মতো compiled language নয়। তবে এটা তার সোর্স কোড কে compile করে প্রোগ্রামের মধ্যে প্রেরন করে। যা মেশিন সহজে বুঝতে পারে।বেশিরভাগ আধুনিক ওয়েব ব্রাউজার জাভাস্ক্রিপ্ট সমর্থন করে এবং তাদের জাভাস্ক্রিপ্ট ইঞ্জিন রয়েছে। Google Chrome এর নিজস্ব জাভাস্ক্রিপ্ট ইঞ্জিন রয়েছে যার নাম V8। এই ইঞ্জিন যা জাভাস্ক্রিপ্টের কোড বোঝে এবং এটি চালায়।একটি জাভাস্ক্রিপ্ট ইঞ্জিনে, সোর্স কোডটি বেশ কয়েকটি ধাপের মধ্য দিয়ে যায়।
=> কোড কে পাস করে।
=> AST( Abstract Syntax Tree) নামক ডেটা স্ট্রাকচার তৈরি করে
=> জাভাস্ক্রিপ্ট ইঞ্জিন জাভাস্ক্রিপ্ট কোডটিকে মেশিন কোডে রূপান্তর করে।
=> শেষে, সেই বাইট কোডটি সিস্টেম দ্বারা চালিত হয়।

Difference between localStorage and sessionStorage

session Storage এবং local Storage এর মধ্যে পার্থক্য হল যে local Storage ডেটার মেয়াদ শেষ হয় না, session Storage এর সেশন শেষ হলে session Storage এর ডেটা delete হয়ে যায়।

local Storage
=> local Storage এর ডাটা স্থায়ী ভাবে থাকে ।
=> local Storage এর ডাটা মুছে দিতে হয় ।
=> local Storage এর ক্ষমতা 5-10 MB

session Storage
=> session Storage ডাটা স্থায়ী ভাবে থাকে না ।
=> session Storage এর ডাটা মুছে দিতে হয় না ।
=> session Storage এর ক্ষমতা 5MB
